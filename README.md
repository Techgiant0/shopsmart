# ShopSmart - Mini E-Commerce Store

ShopSmart is a responsive, modern mini e-commerce frontend built as a capstone project. It simulates an online shopping experience with features such as product browsing, searching, filtering, and cart management.

## 🌟 Features

- **Product Listing:** Browse products seamlessly with dynamic grids.
- **Search & Filter:** Search products by name, filter by category, and sort by price or name.
- **Product Details Modal:** View expanded details, rating, and larger images.
- **Shopping Cart System:** Add, remove, and adjust quantity of items. Displays total cost.
- **Data Persistence:** Uses `localStorage` to save your Cart and Wishlist items so they survive page reloads.
- **Bonus - Wishlist:** Add your favorite products to a persistent wishlist.
- **Bonus - Checkout Flow:** Proceed to checkout with a coupon system (`DISCOUNT10` applies a 10% discount) and order confirmation logic.
- **Bonus - Dark Mode:** Built-in dark mode toggle for comfortable viewing.
- **Responsive UI:** Works flawlessly on desktop, tablet, and mobile.

## 🛠️ Technologies Used

- React (Functional components, Hooks)
- Vite
- Vanilla CSS with CSS Variables for Dark Mode
- `lucide-react` for beautiful SVG icons

## 🔌 API Used

This project fetches mock data from the [Fake Store API](https://fakestoreapi.com/).

## 🚀 How to Run the Project

1. **Clone the repository** (if not already cloned)
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Open in browser:**
   Open the Local URL provided in the terminal (usually `http://localhost:5173/`).
